Features · Agentic Framework

The BESSER Agentic Framework.

Model, build, and deploy AI agents the BESSER way — low-code, open source, and LLM-agnostic. The Agentic Framework (BAF) brings agents into the same model-driven workflow you already use.

Read the docs View on GitHub

Define agents as models

Describe agents, their states, and behaviours with the same model-driven approach as the rest of BESSER — no boilerplate.

Multi-agent & A2A

Compose multiple agents that collaborate, with agent-to-agent (A2A) interoperability built in.

LLM-agnostic

Plug in the language models you prefer — local or hosted — without rewriting your agent logic.

Tools & integrations

Give agents tools and connect them to external systems, APIs, and data sources.

Run & deploy

Generate and deploy your agents the same way you ship the rest of your BESSER-built software.

Free & open source

BAF is part of the open-source BESSER ecosystem — inspect it, fork it, and build on it.

An open-source low-code platform for building smart, AI-enhanced applications faster.
